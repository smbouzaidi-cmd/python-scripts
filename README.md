# Task Manager Python Script

Python script to manage daily tasks by creating, reading, and organizing them in files and folders.

## Purpose
This project allows users to easily track their tasks. It can:
- Create a file containing tasks
- Read and display tasks
- Rename files
- Move files to specific folders

## Requirements
- Python 3.x
- os (built-in)
- shutil (built-in)

## Usage
1. Define your tasks in the script as a list:
```python
tasks = ["Check emails", "Prepare report", "Team meeting", "Review code"]
